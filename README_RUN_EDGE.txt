# Cách mở tool (Edge) - 1 click

## Cách 1 (khuyến nghị, không bao giờ lỗi ảnh):
1) Double-click: RUN_EDGE_1CLICK.bat
2) Edge sẽ tự mở đúng link: http://localhost:8000/index.html

## Cách 2 (fallback):
Double-click: RUN_EDGE_FALLBACK.bat

Lưu ý:
- Cách 1 cần Python. Nếu máy chưa có Python, cài Python từ Microsoft Store hoặc python.org.
